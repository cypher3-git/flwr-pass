# PASS + GTG-Shapley 联邦学习系统交互流程图

    %% ========================================

    %% ========================================

    %% ========================================
    RoundStart --> Phase1[阶段1: 客户端训练]
    
    Phase1 --> ServerSend1[服务器广播训练消息]
    ServerSend1 --> SendModel["发送全局模型 θ(t)"]
    SendModel --> SendConfig["发送训练配置<br/>lr, local_epochs"]
    
    SendConfig --> ClientReceive[所有客户端接收消息]
    ClientReceive --> ClientType{客户端类型?}

    NormalTrain --> TrainEpochs[训练 local_epochs 轮]
    TrainEpochs --> SGDUpdate["SGD优化: θi = θ(t) - lr·∇L"]

    AddNoise --> AFRReturn[模拟零贡献]
    
    ClientType -->|SFR客户端<br/>ID=7| SFRAttack[错误数据训练<br/>当前禁用]
    SFRAttack --> WrongData[在错误数据集上训练<br/>MNIST代替CIFAR-10]

    ClientReturn1 --> ReturnParams["上传: θi_new, num_examples"]

    %% ========================================
    %% 阶段2: FedAvg聚合

    ReturnMetrics --> Phase2[阶段2: FedAvg聚合]

    CalcWeight --> Aggregate["聚合: θ(t+1) = Σ wi · θi_new"]

    SaveState --> SaveBase["保存基础模型: M(t) = θ(t)"]

    SendDeltas --> SendDataSizes["发送数据集大小 |Di|(t-1)"]

    CalcV0 --> CalcVN["计算 vn = V(M(t))"]

    CheckBetween -->|否| MCLoop[蒙特卡洛采样循环]

    InitSV --> SamplingRound[第 k 轮采样]
    
    SamplingRound --> GuidedSampling[引导采样生成排列]
    GuidedSampling --> FrontM[前m个位置循环轮替]
    FrontM --> RemainRandom[剩余n-m个位置随机排列]
    RemainRandom --> Permutation["得到排列 π = (π1, π2, ..., πn)"]
    
    Permutation --> TraverseLoop[遍历排列计算边际贡献]
    TraverseLoop --> InitPrevAcc["初始化 v0 = V(M(t-1))"]
    InitPrevAcc --> ForEachClient{"对每个客户端 πj"}
    
    ForEachClient --> WithinTrunc{轮内截断判断}
    WithinTrunc --> CheckWithin{"|vn - vj-1| < εi?"}
    
    CheckWithin -->|是| TruncWithin["触发轮内截断<br/>marginal = 0"]
    CheckWithin -->|否| ReconstructModel[重构子模型]
    
    ReconstructModel --> SubsetS["子集 S = (π1, ..., πj)"]
    SubsetS --> ReconFormula["重构公式: Ms = M(t-1) + Σ(|Di|/|Ds|)·Δi"]
    ReconFormula --> EvalSubset["评估子模型: vj = V(Ms)"]
    EvalSubset --> CalcMarginal["计算边际贡献<br/>marginal = vj - vj-1"]
    
    TruncWithin --> UpdateSV[增量更新Shapley值]
    CalcMarginal --> UpdateSV
    UpdateSV --> SVFormula["φi = ((k-1)/k)·φi + (1/k)·marginal"]
    
    SVFormula --> NextClient{还有客户端?}
    NextClient -->|是| ForEachClient
    NextClient -->|否| CheckConverge{收敛判定}
    
    CheckConverge --> CalcCV["计算变异系数<br/>CV = std/mean"]

    CVCheck -->|是| Converged[提前收敛]
    CVCheck -->|否| CheckMaxRounds{"k < max_rounds?"}
    
    CheckMaxRounds -->|是| SamplingRound
    CheckMaxRounds -->|否| MaxReached[达到最大采样轮数]
    
    Converged --> ReturnSV[返回Shapley值]
    MaxReached --> ReturnSV
    TruncBetween --> ReturnSV
    
    ReturnSV --> ClientReturn3[客户端上传Shapley值矩阵]

    UploadSV --> Phase4[阶段4: 贡献评估与剔除]
    SkipAudit --> Phase4
    
    Phase4 --> CollectSV[服务器收集所有Shapley值]
    CollectSV --> BuildMatrix["构建Shapley值矩阵<br/>SVij = 客户端j对客户端i的评估"]
    
    BuildMatrix --> CalcMeanSV[计算每个客户端的平均Shapley值]
    CalcMeanSV --> MeanFormula["mean_SVi = (1/(N-1))·Σj SVij"]
    
    MeanFormula --> UpdateCumulative[更新累积Shapley值]
    UpdateCumulative --> CumFormula["cumulative_SVi += mean_SVi"]
    
    CumFormula --> RankClients[基于累积值排名客户端]
    RankClients --> SortBySV["按cumulative_SVi升序排序"]
    SortBySV --> CalcRankScore["计算排名分数<br/>rank_score = rank/(n-1)"]

## 流程图说明

### 阶段划分与颜色标识

- **蓝色**: 系统初始化阶段
- **橙色**: 阶段1 - 客户端训练
- **绿色**: 阶段2 - FedAvg聚合
- **紫色**: 阶段3 - GTG-Shapley审计
- **红色**: 阶段4 - 贡献评估与剔除
- **青色**: 评估与保存
- **深红色**: 搭便车者攻击和剔除

### 关键技术点

#### 1. GTG-Shapley三层截断机制

- **轮间截断**: `|V(M^(t+1)) - V(M^(t))| ≤ ε_b` → 跳过本轮审计
- **轮内截断**: `|V_N - V_{j-1}| < ε_i` → 后续边际贡献为0
- **引导采样**: 前m个位置循环轮替,剩余位置随机排列

#### 2. 子模型重构公式

```
M̃_S = M^(t) + Σ_{i∈S} (|D_i|/|D_S|) * Δ_i
```

避免重复训练,直接从梯度更新重构子模型。

#### 3. Shapley值增量更新

```
φ_i^(k) = ((k-1)/k) * φ_i^(k-1) + (1/k) * marginal
```

在线更新,无需存储所有历史边际贡献。

#### 4. 贡献分数计算

```
cumulative_SV_i += mean(SV_i)
rank_score = rank / (n-1)
score_i = α * old_score_i + (1-α) * rank_score
```

基于累积Shapley值的相对排名,使用移动平均平滑。

#### 5. 剔除判定

```
threshold = 1 / (β * N)
if score_i < threshold → ELIMINATED
```

### 数据流向

1. **训练阶段**: `θ^t → 客户端 → θ_i^new`
2. **聚合阶段**: `{θ_i^new} → FedAvg → θ^(t+1), {Δ_i}`
3. **审计阶段**: `{M^t, Δ_i, |D_i|} → GTG-Shapley → {SV_ij}`
4. **剔除阶段**: `{SV_ij} → 排名评分 → 剔除决策`

### 搭便车者检测

- **AFR客户端(ID=2)**: 返回原始参数+小噪声,模拟零贡献
- **检测机制**: Shapley值低 → 累积排名低 → 贡献分数低 → 被剔除

### 性能优化

- **缓存评估结果**: 避免重复评估相同子集
- **收敛判定**: 变异系数 < threshold 提前停止采样
- **共享排列**: 一次排列计算所有客户端的边际贡献
